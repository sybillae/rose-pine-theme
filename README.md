## Installation

1. Open Phoenix Code
2. Under `File`, click `Extension Manager`
3. Select the `Themes` tab and search for Rosé Pine, Install

## Manual Installation
<sub>cause idk if its just me or its not showing up in the official ph.dev store</sub>

1. Download the v1.0.2 version .zip (here [rose-pine-theme.zip](https://github.com/user-attachments/files/24138603/rose-pine-theme.zip))

2. Navigate to `C:\Users\NAME\AppData\Local\io.phcode\assets\extensions\user`
3. Here you have all the themes and plugins you've installed. Extract rose-pine-theme.zip here.
4. Now inside Phoenix Code, press `View` > `Themes...`, and Rose Pine should be available from your Themes list.
